Projecte Claroline

https://github.com/claroline/Claroline

Instal·lació del requisits

sudo apt-get install php-curl php-fileinfo php-gd php-intl php-mbstring php-mcrypt php-xml php-json php-zip ffmpeg composer

Instal·lació de nodejs

Instal·lem el nodejs des afegint-hi un nou repositori: https://nodejs.org/en/download/package-manager/#debian-and-ubuntu-based-linux-distributions

Alternatively, for Node.js 9: 

curl -sL https://deb.nodesource.com/setup_9.x | sudo -E bash -
sudo apt-get install -y nodejs

Instal·lació

Baixem la última release a /var/www/claroline.local: https://github.com/claroline/Claroline/releases/latest

Descomprimim el contingut de l'arxiu dins de l'arrel de la nostra web.

Executem el script de configuració (hem de tenir creada la base de dades): 

php scripts/configure.php

Anem al directori del projecte /var/www/claroline.local i executem el composer per a que es baixi les dependències. 

composer install
sudo composer sync-dev

Si tot ha anat bé ara ja podem veure la nostra web a: 

http://claroline.local/web/app.php
